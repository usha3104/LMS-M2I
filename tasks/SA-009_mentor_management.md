# SA-009: Mentor Management

## Overview
Super Admin can manage mentors across all colleges in the system.

## Features

### Create Mentor Profile
- Add mentor name
- Add mentor email
- Add mentor phone number
- Add mentor qualification
- Add mentor experience
- Upload mentor profile photo

### Assign College to Mentor
- Select mentor from list
- Select college to assign
- Confirm assignment

### Assign Batch to Mentor
- Select mentor from list
- Select batch to assign
- Confirm assignment

### Manage Mentor Permissions
- View assigned colleges
- View assigned batches
- View student details
- Update student attendance
- Update student marks

### Activate / Deactivate Mentor
- View all mentors
- Search mentor by name/email
- Activate mentor account
- Deactivate mentor account
