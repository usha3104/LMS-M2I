# Story: Super Admin Complete Workflow (Backend)

## Type
Story

## Description
Implement the Express.js backend capabilities required for the Super Admin complete workflow shown in `diagrams/01_Super_Admin_Flow.puml`, including authentication, dashboard access, and management modules.

## Scope
- Super Admin authentication with email/password and OTP
- Super Admin dashboard access
- College management
- NetPy Admin management
- Course management
- Batch management
- Student applications processing
- Reports and monitoring

## Acceptance Criteria
- Super Admin can authenticate using email/password and OTP.
- Authenticated Super Admin can access dashboard endpoints.
- All management modules from the diagram have backend endpoints and service logic.
- All write actions are authorized for Super Admin only.
- API errors are consistent and validated.
